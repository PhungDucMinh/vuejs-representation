Step 3: Identify what is data, what is props

  • Is it passed in from a parent via props? If so, it probably isn’t data.
  • Does it remain unchanged over time? If so, it probably isn’t data.
  • Can you compute it based on any other data or props in your component? If so, it isn’t data.
